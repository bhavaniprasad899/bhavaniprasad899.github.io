CyberArena – Interactive Cybersecurity Learning Platform:

CyberArena is a web-based cybersecurity learning platform designed to help students and beginners understand ethical hacking concepts through interactive games, quizzes, and command-line simulations. The project focuses on practical learning with a clean, professional, hacker-themed interface.

Features:

- Interactive cybersecurity games (Guess the Number, Mini Carrom)
- Command Line Simulator for practicing Linux and hacking commands
- Multiple quiz levels (Level 1 to Level 6)
- MCQ-based learning with instant interaction
- Dark, terminal-style user interface
- Responsive design for mobile and desktop
- Lightweight and fast (no backend required)

Project Structure:

CyberArena/
├── index.html – Home page
├── terminal.html – Command Line Simulator
├── task.html – Quiz and level tasks
├── guess.html – Guess the Number game
├── README.md – Project documentation

Technologies Used:

- HTML5
- CSS3
- JavaScript (Vanilla)
- GitHub Pages for hosting

Purpose of the Project:

- Provide a practical approach to learning cybersecurity
- Help students understand ethical hacking concepts safely
- Build a strong academic and professional portfolio project
- Prepare freshers for entry-level cybersecurity roles
